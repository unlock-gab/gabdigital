- Show icon on card: ON / OFF
or
- Enable icon: true / false

The icon field must become optional.

- do not show the icon block

- If image exists and icon is OFF → show image only
- If image exists and icon is ON → optionally show icon overlay

5) ADMIN UI IMPROVEMENT
In the admin category/service form:
- add a toggle:
  "Show icon on card"

- optionally disable the icon dropdown visually when OFF

The current square icon block in the center of the image area should be removed when icon display is disabled.
I want a cleaner, image-first design.

7) PUBLIC WEBSITE RESULT
On the public cards, when I disable the icon:
- the card should look cleaner
- no floating icon box
- no default icon overlay
- just the image, title, description, and button

DELIVERABLE:
